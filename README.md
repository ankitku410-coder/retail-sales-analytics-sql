# Retail Sales Analytics SQL

A hands-on Data Engineering project built using Microsoft SQL Server. This project simulates a retail company's database and demonstrates database design, SQL queries, and business analytics.

- SQL Server Management Studio (SSMS)
- Visual Studio Code
- Git
- GitHub
